## Descripción

Este proyecto consiste en una **plataforma web** diseñada para mejorar la gestión de solicitudes de soporte técnico dentro de la empresa **123digit@l**. A través de esta plataforma, se optimizan procesos como el registro de solicitudes, la asignación de actividades a colaboradores, el seguimiento del estado de las solicitudes y la generación de reportes. 

La solución implementa una arquitectura robusta basada en **Modelo-Vista-Controlador (MVC)** y utiliza tecnologías modernas para garantizar escalabilidad, seguridad y una experiencia de usuario fluida.

## Funcionalidades Principales

- **Gestión de Solicitudes:**
  - Registro de nuevas solicitudes por parte de los clientes.
  - Consulta del estado de las solicitudes.
  - Actualización del estado de solicitudes por colaboradores.

- **Gestión de Actividades:**
  - Asignación de actividades a colaboradores específicos.
  - Visualización y seguimiento de actividades asignadas.

- **Gestión de Usuarios y Roles:**
  - Registro y autenticación de usuarios.
  - Gestión de roles (Administrador, Colaborador, Cliente).

- **Dashboard y Reportes:**
  - Generación de gráficos estadísticos sobre solicitudes.
  - Visualización de datos clave para la toma de decisiones.

- **Notificaciones:**
  - Envío automático de correos electrónicos para registro de cuentas y cambios en solicitudes.

## Tecnologías Utilizadas

### Backend
- **Java** (Servlets, Beans)
- **PostgreSQL** (Base de datos relacional)
- **HikariCP** (Gestión de conexiones)
- **Patrones de diseño:** Singleton, DAO, DTO, Facade, Factory

### Frontend
- **HTML5, CSS3, XHTML**
- **PrimeFaces** (Framework para componentes UI)
- **TailwindCSS** (Estilización moderna)

### Herramientas de Desarrollo
- **IDE:** IntelliJ IDEA / Eclipse
- **Control de Versiones:** Git & GitHub
- **Diagramación:** Draw.io (Diagramas UML)
